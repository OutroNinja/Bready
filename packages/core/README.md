# Bready Core
Simple functions to facilitate development

## Installation
To download the module, run the following command
```bash
npm install @bready/core
```

## Usage
### CommonJS
```js
const { capitalize } = require("@bready/core");
```

### ESM
```js
import { capitalize } from "@bready/core";
```
