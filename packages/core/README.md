<a name="install"></a>
## Install
To install `@bready/core`, simply use npm:
```bash
npm install @bready/core
```

<a name="usage"></a>
## Usage
### CommonJS
```js
const bd = require("@bready/core");
```

### ESM
```js
import { capitalize, replaceString } from "@bready/core";
```

Default exports are supported:
```js
import bd from "@bready/core";
```
